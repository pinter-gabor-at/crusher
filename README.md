# Minecraft Fabric Crusher Mod

Similar to a furnace, but crusher.

Recipes:

Stone -> Gravel

Gravel -> Sand

Blaze rod -> Blaze powder

Concrete -> Concrete powder

Bone -> Bone meal

And why not make a machine too to make the opposite? A Compressor.
